Equal height - top

  • In features list (like 'features-small'), text may be uneven, it may distort looks.
  • Bringing all lists to equal height will make it fine.
  • Needs function, you may place it in custom.js, in document ready. 
    •        
function equalHeight(group) {
tallest = 0;
group.each(function() {
thisHeight = $(this).height();
if(thisHeight > tallest) {
tallest = thisHeight;
}
});
group.height(tallest);
}
equalHeight($(".equal_height li"));

Get started - top

  • Download the final zip file from your downloads page.
  • Unzip the final zip file; you should now have a final download folder.
  • Do NOT upload the final download folder. Open up the final download folder.
  • Browse the contents and find the actual themes folder (inside 'HTML and CSS' folder).
  • Upload ONLY the selected (color / variation) theme folder.
  • Regarding uploading theme to your hosting account - go through the help / support of your hosting service provider. Drop a ticket to them if any query :)

Customization tips - top

  • While removing any jquery library attached...

    If you are removing any jquery library attached, remember to remove its function form custom.js (if any), otherwise it will affect page display in many browsers esp. IE. If you need to attach some jquery element to particular page only, it is best practice to add its jquery, css, function etc. in head section of that page (as it is done for sliders, prettyphoto etc.)

    Following jqueries are attached to all pages in template.

    • bootstrap.js
    • ddsmoothmenu.js
    • jquery.validate.js
    • jquery.form.js
    • jquery.easing.1.3.js
    • custom.js
