How make twitter feed work?

For buyers of 'version 1.07' and above

  • Setup a Twitter Application
  • Authenticate the Twitter Feed
  • Provide Twitter Username
  • Twitter Still not Working

Older version buyer please refer 'How can I update twitter?'

Setup a Twitter Application - top

  • Open - https://dev.twitter.com/apps/
  • Sign in with your Twitter username and password.
  • Click on 'Create a new application' and provide all application details.
  • Put up any name and description, just do not use 'Twitter' in the name.
  • Website field - name of your main website, not necessary to put up name of website where you are calling Twitter feed/feeds.
  • Callback URL field - can be left blank
  • Select checkbox - Yes, I agree
  • Enter CAPTCHA
  • Click on button - Create your Twitter application
  • On the next details screen, click 'Create my access token'.
  • Refresh the page after a few seconds if it doesn't appear automatically.
  • You will get Consumer key, Consumer secret, Access token and Access token secret as shown below.

Authenticate the Twitter Feed - top

  • Give your Consumer key, Consumer secret, Access token and Access token secret in 'index.php' (inside 'twitter' folder). 
      
    // Your Twitter App Consumer Key
	private $consumer_key = 'YOUR_CONSUMER_KEY';

	// Your Twitter App Consumer Secret
	private $consumer_secret = 'YOUR_CONSUMER_SECRET';

	// Your Twitter App Access Token
	private $user_token = 'YOUR_ACCESS_TOKEN';

	// Your Twitter App Access Token Secret
	private $user_secret = 'YOUR_ACCESS_TOKEN_SECRET';

Provide Twitter Username - top

  • Specify your username and number of tweets (count) you need to show, in the twitter function.
    This function is located in 'custom.js' 
    $(document).ready(function () {
//twitter feed
$(".twitter").tweet({
	username: "envato",
	count: 1,
	loading_text: "loading tweet...",
	join_text: "auto",
	auto_join_text_default: "",      
	auto_join_text_ed: "",                   
	auto_join_text_ing: "",               
	auto_join_text_reply: "",     
	auto_join_text_url: "", 
	modpath:'twitter/'
});
});

Twitter Still not Working - top

  • It needs to 'upload' all files to see the twitter feed. Do not works locally.

  • Try to give proper path for 'twitter folder' via the modpath option , relative to your domain (like 'http://yoursitename.com/twitter/').

    As shown below 
    $(document).ready(function () {
//twitter feed
$(".twitter").tweet({
join_text: "auto",
auto_join_text_reply: "",   
modpath:'http://yoursitename.com/twitter/',
username: "envato",
count: 1,
loading_text: "loading tweet..."
});
});
